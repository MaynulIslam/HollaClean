# 🏠 HollaClean - Personal Home Cleaning Marketplace

> Connect homeowners with trusted residential cleaning professionals for personal home care

HollaClean is a complete, production-ready Progressive Web App (PWA) marketplace that enables homeowners to find and book **personal residential cleaning services** for their homes, apartments, and condos. Built specifically for the home cleaning industry with modern web technologies and optimized for mobile devices.

## ✨ What Makes HollaClean Special

**Personal Home Cleaning Focus:**
- 🏡 Residential properties only (homes, apartments, condos)
- 👨‍👩‍👧‍👦 Personal household cleaning services
- 🧹 Regular maintenance, deep cleaning, move-in/out
- 🧺 Laundry, ironing, and household tasks
- 🪟 Specialized home services (windows, carpets)

**NOT for commercial/office cleaning** - Built specifically for homeowners seeking personal cleaning help.

## ✨ Key Features

**For Homeowners:**
- 🔍 Browse verified residential cleaning professionals
- 📅 Book personal home cleaning in under 60 seconds
- 💳 Secure payment processing via Stripe
- ⭐ Rate and review your home cleaner
- 📱 Track your home cleaning appointments
- 🏠 Manage multiple properties/addresses

**For Residential Cleaning Professionals:**
- 💼 Get more homeowner clients automatically
- 📊 Manage all household cleaning jobs
- 💰 Track earnings from home cleaning services
- 🌟 Build reputation in residential cleaning
- 📲 Accept/decline home cleaning requests
- 🧹 Showcase your home cleaning specialties

**For Platform Owners:**
- 💵 Automated 20% commission on all home cleanings
- 📈 Complete admin dashboard with analytics
- 👥 Manage homeowners and cleaning professionals
- 💳 Integrated payment processing
- 📊 Track residential cleaning market trends

## 🚀 Tech Stack

- **Frontend:** React 18, Tailwind CSS, Lucide Icons
- **Build Tool:** Vite
- **Payments:** Stripe (ready to integrate)
- **Storage:** Browser Storage API
- **PWA:** Service Worker, Web Manifest
- **Deployment:** Vercel-ready

## 📱 Mobile-First Design

- Installable as native app on iOS and Android
- Touch-optimized UI for easy booking
- Bottom navigation for homeowner convenience
- Offline functionality
- Perfect for on-the-go home management

## 💰 Business Model for Personal Cleaning Services

- Platform earns 20% commission per home cleaning booking
- Stripe processes payments (2.9% + $0.30)
- Automated payment splits to home cleaners (80%)
- Complete revenue tracking built-in

**Example:** $100 home cleaning = $16.80 platform profit after Stripe fees

**Typical Home Cleaning Rates (Ontario):**
- Regular cleaning: $25-35/hour
- Deep cleaning: $30-50/hour
- Move-in/out: $150-400 per job
- Average booking: $80-120

## 🎯 Quick Start
```bash
# Clone the repository
git clone https://github.com/yourusername/hollaclean.git

# Navigate to project
cd hollaclean

# Install dependencies
npm install

# Start development server
npm run dev
```

Open `http://localhost:3000` - Your personal home cleaning marketplace is running! 🎉
