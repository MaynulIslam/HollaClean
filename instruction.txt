for stripe:

How it works now
With server running (cd server && node index.js):

Creates a real Stripe PaymentIntent via your backend
Mounts a secure Stripe Card Element (iframe — Stripe handles card data directly)
Calls stripe.confirmCardPayment() with the real clientSecret
Payments appear in your Stripe Test Dashboard
Use test card 4242 4242 4242 4242, any future expiry, any CVC
Without server (demo mode):

Falls back to the manual card form with "Demo Mode" banner
Simulates payment success after 2 seconds
Behavior identical to before
To test real payments

cd server && node index.js    # Terminal 1 — starts Express on port 3001
cd .. && npm run dev           # Terminal 2 — starts Vite on port 3000
Then go through the payment flow — you should see the charge on your Stripe test dashboard.
